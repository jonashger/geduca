# Para iniciar o desenvolvimento

Baixar Sencha CMD
https://www.sencha.com/products/extjs/cmd-download/

Adicionar as variaveis de ambiente do sencha no windows. 
Usuario  
ex: Path - C:\Users\users\bin\Sencha\Cmd\6.5.1.240
Sistema
ex: Path - C:\Users\users\bin\Sencha\Cmd\6.5.1.240\sencha.exe

Documentação do sencha a ser utilizada;
http://docs.sencha.com/extjs/6.5.1/modern/Ext.html

Eclipse EE para IDE
http://www.eclipse.org/downloads/download.php?file=/technology/epp/downloads/release/kepler/SR2/eclipse-jee-kepler-SR2-win32-x86_64.zip

Iniciando a aplicacão
Entrar na pasta do git e pelo cmd/powershel. Dar o seguinte comando. "sencha web start" 
abrir o site com o ip indicado
